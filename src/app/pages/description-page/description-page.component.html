Описание курса
Сегодня игры стали постоянным спутником человека. С одной стороны, распространение мобильных
устройств позволило людям посвящать играм больше времени, с другой — геймификация проникает во
все сферы жизни, и игровой софт требуется уже далеко не только для развлекательных целей. В ходе
курса вы познакомитесь с главными элементами разработки: изучите игровой ландшафт, дизайн игрового
мира, VR и AR в Unity, 3D-графику, звуковые эффекты и музыку.
